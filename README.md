# Operations - репозиторий для запуска проекта в различных средах

## Структура репозитория

- stage - для запуска проекта в различных средах из готовых образов (требует установки Docker, Make)
- local - для локального запуска проекта для разработки и тестирования (требует установки Go, Node, npm и др.)

## Примечание

Для проверки работоспособности проекта и демонстрации используйте stage.
